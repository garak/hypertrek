Download

In questa pagina viene spiegato come ricreare sul proprio computer una copia di HyperTrek conforme con quella online. Prerequisiti:

  • File System che supporta i nomi lunghi, i nomi dei file di HyperTrek non contengono spazi e non usano lettere maiuscole.
  • Scompattatore ZIP oppure RAR (lo scompattatore RAR è disponibile gratuitamente per molte piattaforme su www.rarlab.com, inoltre molti programmi decomprimono i file RAR).
  • Una directory vuota, che diventerà la directory root di HyperTrek.

È consigliabile installare anche il font TrueType Aero; alcune vecchie versioni di Netscape interpretano erroneamente questo file come eseguibile, si tratta invece di un file TrueType con estensione TTF.

Ogni archivio contiene il path dei file riferito alla root di HyperTrek, quindi ciascun file va scompattato dalla root di HyperTrek. Quando si installa una versione completa, è sempre meglio cancellare la directory prima di scompattare l'archivio nuovo.
